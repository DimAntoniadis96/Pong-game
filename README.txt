

			_______PONG_______

Αυτό είναι ένα 2D pong-style παιχνίδι που δημιουργήθηκε με Python και pygame.

Πώς να το παίξεις:
1. Κάνε download και run το αρχείο PONG_GAME_EAP.exe
2. Το παιχνίδι ξεκινάει άμεσα – δεν απαιτεί εγκατάσταση

Χειρισμός:
- Παίκτης 1: ↑ (πάνω), ↓ (κάτω) 
- Παίκτης 2: W (πάνω), S (κάτω)

Δυνατότητες:
- Μετά τους 4 πόντους, οι ρακέτες μπορούν να κινούνται και οριζόντια (δεξιά–αριστερά),
  προσθέτοντας επιπλέον πρόκληση στο παιχνίδι.
- Δυνατότητα για πλήρη οθόνη.
- Κάνοντας κλικ επάνω στο εικονίδιο του ήχου γίνεται mute.


Υπάρχουν εφέ ήχου, μουσική φόντου και γραμματοσειρές ειδικά ενσωματωμένες.
Δεν απαιτείται εγκατάσταση ή εξωτερικό πρόγραμμα. Το αρχείο είναι αυτόνομο.
